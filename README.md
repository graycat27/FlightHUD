## 前提環境
Minecraft JavaEdition
Forge1.16.4（各releaseのバージョン情報が最新です）

## 操作
エリトラ滑空中・落下中・スペクテイター の各状態で自動的に表示されます。
表示が不要な場合、[H]キーでOFFにできます（スペクテイターの時とか…？）
